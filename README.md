# builtbyjess.dev — Portfolio Site

Personal portfolio site for Jessica Gonzales, software developer focused on cloud engineering, DevOps, and platform engineering.

Live site: [builtbyjess.dev](https://www.builtbyjess.dev)

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- Hosted via GitHub Pages
- Custom domain via CNAME

## Running Locally

Open `index.html` in IntelliJ IDEA and click the browser icon in the top-right corner of the editor. IntelliJ's built-in server will serve the site at `localhost:63342` — this handles the background video correctly, which won't play over a plain `file://` URL.

## Pages

| Page | Description |
|---|---|
| `index.html` | Landing page — hero, about, projects, contact |
| `resume.html` | Resume page with downloadable PDF |
